# DevCLI

DevCLI is a command-line utility tool built using Node.js and TypeScript.

It demonstrates Object-Oriented Programming, API integration, and CLI command creation.

---

## Features

• 10 CLI commands  
• Built using TypeScript  
• Object-Oriented architecture  
• API integration  
• File utilities  
• System utilities  

---

## APIs Used

GitHub API  
Quotable API  
Weather API  
Joke API  

---

## Installation

Clone the repository

```bash
git clone https://github.com/harshcoder-harsh/dev_cli.git
```

Install dependencies

```bash
npm install
```

Run CLI

```bash
npm start
```

---

## Commands

**greet <name>**
Greets the user.

**github <username>**
Shows GitHub user information.

**weather <city>**
Displays weather for a city.

**quote**
Returns a random quote.

**joke**
Displays a random joke.

**fileinfo <filename>**
Shows file size and creation date.

**date**
Displays current date and time.

**random**
Generates a random number.

**calc \<a> \<b>**
Adds two numbers.

**system**
Shows system information.

---
